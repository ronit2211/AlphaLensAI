# AlphaLens AI


## Overview

AlphaLensAI is an AI-powered equity research and stock analytics platform that combines quantitative screening, technical analysis, portfolio analytics, and Gemini AI-generated investment insights.

The platform enables investors to analyze stocks, compare opportunities, generate AI watchlists, and evaluate portfolio performance using a unified dashboard.

---

## Features

### Stock Analysis

* Technical indicator calculation
* RSI (Relative Strength Index)
* MACD (Moving Average Convergence Divergence)
* SMA 20 and SMA 50
* Trend Detection
* AI-generated stock research reports

### Stock Comparison

* Compare multiple stocks
* Alpha Score comparison
* Risk assessment
* Trend comparison
* AI-generated comparison reports

### AI Watchlist

* Automated stock ranking
* Alpha Score engine
* Opportunity identification
* AI-generated watchlist commentary

### Portfolio Analytics

* Portfolio return tracking
* Volatility analysis
* Sharpe Ratio monitoring
* Portfolio health assessment
* Risk evaluation
